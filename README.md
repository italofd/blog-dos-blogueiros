### Features

* Listar posts
* Mostrar quantidade de posts
* Mostar todos os likes recebidos em todos os posts feito por determinado usuario
* Possibilidade de like no post
* Criar novo post
* Sistema de login e register
* Possibilidade de upar uma imagem
